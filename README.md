# Doc2Table

## 项目介绍

Doc2Table 是一个基于 Flask 的文档处理工具，用于将各种格式的文档（如 Word、Markdown、文本文件）中的数据提取并填充到表格模板中。

### 主要功能
- 支持多种文件格式的上传和处理（xlsx, docx, md, txt）
- 提供表格模板上传功能
- 允许用户输入处理需求
- 自动处理数据并生成结果文件
- 提供结果文件的下载功能

### 技术栈
- Python 3
- Flask Web 框架
- 各种文件处理脚本（位于 Scripts 目录）

## 部署启动步骤

1. **运行 setup.py**
   - 在项目根目录下执行：
     ```
     python setup.py
     ```

2. **运行 app.py**
   - 在项目根目录下执行：
     ```
     python app.py
     ```

3. **打开浏览器**
   - 在浏览器中访问：`http://localhost:5000`

## 使用说明

1. 在网页界面中：
   - 上传一个或多个数据文档（支持 xlsx, docx, md, txt 格式）
   - 上传表格模板文件
   - 输入处理需求
   - 点击「处理」按钮

2. 处理完成后，系统会自动生成结果文件并提供下载链接。
